# AdBlocker
專題理念：在上網時經常遇到許多煩人的廣告讓頁面變得亂七八糟，希望透過所學來創造一個可以讓使用者下載的AdBlocker擴展包，讓使用者簡單輕鬆的擺脫廣告的折磨
基本流程：1. 使用者可以透過一個由github生成的網址來抵達我們的下載介面，點按"下載擴充功能"即可下載擴充包
         2. 將擴充胞解壓縮，到瀏覽器點及右上角的三顆點點，並點按"擴充功能"中的"管理擴充功能"(要記得打開開發者模式)
         3. 點載入解壓縮即可成功啟用我們的AdBlocker，好好享受無煩人廣告的瀏覽器吧！

基本功能：1. manifest.json
            為擴充功能的核心，負責註冊 rules.json 給瀏覽器，並告訴瀏覽器在網頁載入時要執行 content.js
         2. rules.json
            負責阻擋廣告的腳本，用蒐集的方式排除多項廣告
         3. content.js
            負責清除廣告阻擋留下的div，讓版面看起來乾淨
         4. popup.html/css
            點擊小圖示時顯示的樣子
         5. popup.js
            協助popup.js 和 content.js 之間的溝通
